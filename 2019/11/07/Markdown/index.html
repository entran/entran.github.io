Markdown标题

标题使用

Markdown标题有两种格式
一级标题 用两个=
二级标题 用两个-

tRan[^name]
[^name]: 英文名

例：

  1. 第一项
  2. 第二项
  3. 第三项
列表嵌套

只需在子列表前加上四个空格即实现嵌套

例：

  1. 第一项
    • 第一项嵌套

Markdown区块

区块使用

区块引用实在段落开头加 > ，紧跟着一个空格

例：

最外层

第二层

第三层

区块中使用列表

直接加在区块符号后面

链接名称
或
<链接地址>

例:
tRan[1]
tRan[2]
[1]: https://www.tranning888.xyz
[2]: https://www.tranning888.xyz

Markdown图片

图片使用

例:
| 表头 | 表头 |
| —- | —- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |

对齐方式

-: 右对齐
:- 左对齐
:-: 居中对齐

| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

例：
| 左对齐 | 右对齐 | 居中对齐 |
| :—–| —-: | :—-: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
